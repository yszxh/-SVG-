三角函数挑战

单位圆的性质

什么是单位圆？

单位圆是一个半径为1的圆，其圆心位于坐标原点(0, 0)。在单位圆上的任意点(x, y)都满足：

x² + y² = 1

三角函数的几何意义

对于单位圆上的点(x, y)，如果角θ是从正x轴逆时针方向到该点的角度，那么：

  • cos(θ) = x坐标值
  • sin(θ) = y坐标值
  • tan(θ) = sin(θ) / cos(θ) = y / x

特殊角度的值

一些常见角度的三角函数值：

  • 0°: (cos=1, sin=0)
  • 30°: (cos=√3/2, sin=1/2)
  • 45°: (cos=√2/2, sin=√2/2)
  • 60°: (cos=1/2, sin=√3/2)
  • 90°: (cos=0, sin=1)

周期性

三角函数具有周期性，对于任意角度θ：

  • sin(θ + 360°) = sin(θ)
  • cos(θ + 360°) = cos(θ)
  • tan(θ + 180°) = tan(θ)
